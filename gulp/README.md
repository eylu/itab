## gulp 压缩合并文件

### 安装

1、安装 node 略

2、安装 gulp

    $ npm install gulp

3、安装其他工具

    $ npm install gulp-util
    $ npm install gulp-concat
    $ npm install gulp-minify-css
    $ npm install gulp-rename
    $ npm install gulp-uglify

### 使用

    $ gulp // 此命令运行默认的 task
    $ gulp minifyCss // 此命令运行名称为 minifyCss 的 task
